# Cloudy Web

Setup-only Next.js application with Tailwind CSS v4 and the complete shadcn Base UI component set.

```bash
pnpm dev
pnpm lint
pnpm build
```

Design tokens live in `src/app/globals.css`. Product screens and backend connections are intentionally deferred.
